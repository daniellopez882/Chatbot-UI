## ğŸš€ Quick Start (Local)

1. **Clone the Repo**
2. **Install Dependencies**: `npm install`
3. **Start Supabase**: `supabase start`
4. **Configure Secrets**: Copy `.env.local.example` to `.env.local` and fill in your Supabase and API keys.
5. **Launch**: `npm run dev`

Your instance will be running at `http://localhost:3000`.
